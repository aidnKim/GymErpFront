# Gym ERP (Frontend)

> **스마트 헬스장 통합 관리 시스템의 사용자 인터페이스(UI) 저장소입니다.**

<br>

## 안내 (Notice)
이 프로젝트의 **전체 아키텍처, DB 설계, 핵심 트러블 슈팅(성능 개선) 내용**은 백엔드 저장소에 상세히 기술되어 있습니다.
**[Backend Repository 바로가기 (Click)](https://github.com/aidnKim/GymErp)**

<br>

## 프론트엔드 기술 스택 (FE Tech Stack)
* **Core:** React 18, JavaScript (ES6+)
* **Styling:** Bootstrap CSS
* **Build Tool:** Vite
* **Communication:** Axios (API 연동)

## 실행 방법 (How to run)
```bash
# 1. 저장소 복제
git clone [https://github.com/사용자아이디/FE저장소명.git](https://github.com/사용자아이디/FE저장소명.git)

# 2. 패키지 설치
npm install

# 3. 개발 서버 실행
npm run dev
